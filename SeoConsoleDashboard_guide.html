📌 이 대시보드로 무엇을 알 수 있나요?

  • 🔍
    실제 유입 검색어 파악
    사용자가 구글/네이버에서 어떤 키워드로 우리 서비스에 들어오는지 확인 가능
  • 📄
    실제 유입 페이지 확인
    사용자가 구글/네이버에서 어떤 페이지로 우리 서비스에 들어오는지 확인 가능
  • 📈
    주차별 증감 추이 모니터링
    당기(이번 주)와 전기(지난 주) 비교로 클릭수·노출수·CTR 변화를 즉시 파악
  • ⭐
    신규 유입 검색어 발굴
    이전 주에는 없었던 새로운 검색어를 NEW 배지로 표시, 유효할 경우 콘텐츠·신규 키워드 발굴에 활용
  • 🚨
    이상 신호 자동 감지
    클릭 급감·CTR 하락·노출 증가-클릭 감소 등 주요 이슈를 인사이트 탭에서 자동으로 알림

🔗 접속 방법

💡
대시보드 접속 URL
담당자에게 공유된 Google Apps Script 웹앱 URL로 접속하면 됩니다.
별도 설치 없이 크롬 브라우저에서 바로 사용 가능합니다.

접속 후 초기 화면

  • 1️⃣
    자동 로딩 시작
    접속 즉시 📊 주차별 요약 탭이 자동으로 로드됩니다 (약 10~30초 소요)
  • 2️⃣
    필터 옵션 초기화
    상단의 도메인 선택·주차 선택 드롭다운이 자동으로 채워집니다
  • 3️⃣
    원하는 탭 클릭
    필터를 설정한 뒤 원하는 탭을 선택하고 조회 버튼을 누르세요
⚠️ 권장 환경: Google Chrome 최신 버전 / 화면 너비 1280px 이상 (와이드 모니터 권장)

🔍 검색어 Top N 탭

화면 구성

  • #
    순위
    당기 클릭수 기준 내림차순
  • 🔑
    검색어 + NEW
    전기(지난 주)에 없었던 신규 유입 검색어에는 NEW 배지가 표시됩니다. 구글은 스팸 키워드 제외 로직을 적용했습니다.
  • 👆
    클릭수 · 노출수 · CTR
    각 지표의 당기 / 전기 / 증감 / 증감률 4개 컬럼으로 표시
    ▲ 빨강 = 증가  |  ▼ 파랑 = 감소

💡 인사이트 탭 자동 분석

🔔 감지되는 인사이트 유형

⚠️ 노출 증가 & 클릭 감소
노출은 늘었으나 클릭이 감소한 검색어 — 검색 순위는 있지만 제목/설명이 매력적이지 않거나 AI 오버뷰에 밀릴 가능성
💼 권장 액션: Meta 태그(title/description) 개선, 랜딩 페이지 점검
📉 클릭 급감 검색어 발생
전주 대비 클릭이 30% 이상 감소한 검색어 — 검색 순위 하락 또는 랜딩 페이지 품질 문제 가능성
💼 권장 액션: 해당 키워드 검색 순위 확인, 랜딩 페이지 품질 점검
⭐ 신규 키워드 유입 발생
이번 주 처음 등장한 신규 검색어 — 새로운 콘텐츠 기회 또는 바이럴·시즈널 트렌드 포착
💼 권장 액션: 관련 콘텐츠·광고 검토, 트렌드성 키워드 여부 확인
🔻 CTR 급락 페이지 발생
클릭률(CTR)이 5%p 이상 하락한 페이지 — 검색 결과 표시 정보(title/snippet)가 사용자 의도와 맞지 않을 가능성
💼 권장 액션: Meta 태그 및 구조화 데이터 점검
🚀 유입 급증 페이지 발생
전주 대비 클릭이 50% 이상 증가한 페이지 — 검색 순위 상승 또는 시즈널 트렌드 반응
💼 권장 액션: 관련 콘텐츠 확대, 내부 링크 강화로 지속 유입 유도
💪 노출 감소 & 클릭 유지
노출은 줄었지만 클릭은 유지되는 검색어 — CTR이 개선되고 있거나 검색량 자체가 줄어든 경우
💼 권장 액션: 검색량 변화 및 순위 점검, 콘텐츠 확충 검토

📂 큐레이션본부 — 카테고리 정비 & 콘텐츠 기획·보완
1
검색어 Top 50에서 클릭 상위 키워드와 NEW 신규 유입 키워드 확인 → 카테고리 개편/정비·콘텐츠 기획 등 활용
2
페이지 Top 50에서 유입 상위 페이지 파악 → 해당 페이지·상품·구매가이드 본문 점검 및 보완 우선순위 결정
3
클릭 많은 검색어 vs. 실제 상품 카테고리 매핑 확인 → 카테고리명·태그 정비 및 대표 상품 노출 강화
🛠️ 서비스기획/모바일본부 — 색인 점검 & 내부 링크 강화 & HTML 태그 보완
1
페이지 Top 50에서 클릭 상위 페이지 목록 확인 → 오류 페이지 여부 점검 및 내부 링크 연결 강화
2
인사이트 탭의 📉 클릭 급감 / 🔻 CTR 급락 카드에서 문제 페이지 포착 → 페이지 색인 및 HTML meta 태그 점검
3
노출은 많으나 CTR이 낮은 페이지 → 검색 결과 스니펫 품질 개선 및 구조화 데이터 점검
4
도메인 비교에서 PC/모바일(www vs m) 도메인 간 성과 격차 확인 → 모바일 페이지 최적화 우선순위 결정
📣 마케팅 — 광고 키워드 & 콘텐츠 소재 발굴 & SEM 최적화
1
검색어 Top 50의 NEW 신규 키워드 → 트렌드성 여부 판단 후 검색 광고·블로그 콘텐츠 주제 발굴
2
검색어 Top 50의 광고 집행 키워드 → 검색 광고 단가·효율 점검 및 최적화 활용
3
인사이트 탭의 🚀 유입 급증 페이지 → 시즈널 트렌드 및 이슈 콘텐츠 포착
4
주차별 요약으로 서비스별 클릭 추이 파악 → 성과 하락 서비스에 마케팅 지원 집중

검색 순위가 높을수록, 검색량이 많을수록 증가.		 잠재 도달 범위 파악

전기 당기 바로 직전 주차의 데이터. 자동으로 설정됨. 비교 기준
WoW 전기 대비 당기 증감률. 주간 성장률 파악

❓ FAQ

Q 데이터는 얼마나 자주 업데이트되나요?
매주 화요일 오전에 자동으로 직전 주(월~일) 데이터를 수집합니다.
예를 들어 4월 1일(화요일)에 접속하면 3월 23일(월)~3월 29일(일) 데이터가 최신입니다.
구글 서치콘솔 특성상 최근 2~3일 데이터는 집계 지연이 있을 수 있어 매주 화요일에 업데이트 합니다.
Q 주차별 요약이 느리게 로딩되는 이유가 있나요?
주차별 요약 탭은 최근 5주치 전체 데이터를 한 번에 분석하므로 데이터 양에 따라 10~30초 소요될 수 있습니다.
로딩 스피너가 표시되는 동안 기다려 주세요.
Q 검색어 Top N에 서비스과 무관한 키워드가 집계되는 이유는?
DPG, 쇼핑지식 등 커뮤니티 내 게시글을 통해 실제 노출/유입이 발생한 검색어이며, 특히 구글에서 두드러집니다.
유해 검색어의 경우 스팸 키워드 제외 로직을 적용하여 주차별로 필터링 업데이트하고 있으나 일부 누락될 수 있습니다.
불필요한 검색어/페이지는 게시글 금칙어, 게시글 신고/삭제 등 작업을 통해 추가 관리 부탁 드립니다.
Q CTR이 갑자기 낮아진 이유는 무엇인가요?
CTR 하락의 주요 원인:
① 검색 결과 순위는 유지되나 Google AI 오버뷰(AI 답변)에 의해 클릭 유도 감소
② 검색 결과 title/description이 사용자 검색 의도와 불일치 (설정해둔 meta 태그와 무관한 내용을 크롤링하는 등)
③ 경쟁 사이트의 리치스니펫(별점·이미지 등) 등장으로 시선 분산
인사이트 탭에서 CTR 급락 페이지를 확인하고 Meta 태그를 점검하세요.
Q 특정 도메인 데이터가 조회되지 않아요.
선택한 주차에 해당 도메인의 데이터가 수집되지 않았거나, 해당 기간에 유입이 없을 수 있습니다.
수집에 문제가 있는 지 퍼포먼스마케팅팀에 문의해주세요.
Q 인사이트에 아무 카드도 나오지 않아요.
인사이트는 전기 데이터와 비교해 이상 신호를 감지합니다.
전기 데이터가 충분하지 않거나(신규 키워드/페이지만 존재), 변화폭이 감지 임계값(예: 클릭 30% 이상 감소)에 미치지 않을 경우
카드가 생성되지 않습니다. 이는 이번 주 데이터가 전반적으로 안정적이라는 긍정적 신호일 수 있습니다.
